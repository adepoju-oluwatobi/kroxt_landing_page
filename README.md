# kroxt Landing Page Setup

This is the official landing page for **kroxt**, your framework-agnostic modular auth engine.

## How to use this folder

The landing page is designed to be **server-less** and ready to go in seconds. Follow these steps to view or deploy it:

### 1. View Locally
Simply double-click the `index.html` file in this folder to open it in your default web browser. 

- **Framework**: No frameworks required! It uses the Tailwind CSS CDN.
- **Interactivity**: Built-in smooth scroll and copy-to-clipboard functionality.

### 2. Customizing Content
To change the text or styling:
1. Open `index.html` in your favorite code editor (like VS Code).
2. Look for the HTML tags like `<h1>` or `<p>`.
3. Save the file and refresh your browser.

### 3. Deployment
This is a static site. You can deploy it for free in seconds using:
- **GitHub Pages**: Upload this folder to a repo and enable Pages.
- **Netlify**: Drag and drop this folder into the Netlify dashboard.
- **Vercel**: Use `vercel deploy` in this directory.
