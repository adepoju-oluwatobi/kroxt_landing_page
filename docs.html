Documentation

Master Kroxt Auth in four simple steps.

1

Installation

Install the core library using your preferred package manager. Kroxt is lightweight and has minimal dependencies.

$ npm install kroxt
2

Configuration

Initialize the library with your chosen adapter and secrets. Here's a standard setup using a MongoDB adapter.

src/lib/auth.ts 
import { createAuth } from "kroxt";
import { mongoAdapter } from "./auth.ts";

export const auth = createAuth({
    adapter: mongoAdapter,
    secret: process.env.JWT_SECRET,
    pepper: process.env.JWT_PEPPER,
    session: {
        expires: "1m",          // Short-lived access tokens
        refreshExpires: "7d",   // Long-lived refresh tokens
    },
});
3

Authentication Flows

Use the initialized auth object to handle basic user operations.

Signup & Login

// Sign up a new user
const { user, accessToken, refreshToken } = await auth.signup(userData, password);

// Log in existing user
const { user, accessToken, refreshToken } = await auth.loginWithPassword(email, password);

Token Refresh

// Generate a new access token using a refresh token
const { accessToken } = await auth.refresh(refreshToken);
4

Securing Routes

Create middleware to protect your API routes using auth.verifyToken.

src/middlewares/auth.middleware.ts 
export const protect = async (req: AuthRequest, res: Response, next: NextFunction) => {
  const token = req.headers.authorization?.split(' ')[1];
  
  try {
    const payload = await auth.verifyToken(token, "access");
    if (!payload) return res.status(401).json({ message: 'Invalid token' });

    req.user = { id: payload.sub, role: payload.role };
    next();
  } catch (error) {
    return res.status(401).json({ message: 'Authorization failed' });
  }
};
5

Adapters & Flexibility

The Core Interface

Implementation is simple. Just provide find and create methods for your database of choice. 

export interface BaseUser {
  id: string;
  email: string;
  passwordHash?: string;
  role?: string;
}

// Supports any extended fields
export type User<TExtended = Record<string, any>> = BaseUser & TExtended;

export interface AuthAdapter<TUser = User> {
  createUser: (data: any) => Promise<TUser>;
  findUserByEmail: (email: string) => Promise<TUser | null>;
  findUserById: (id: string) => Promise<TUser | null>;
}

Mongoose Implementation

A production-ready example using Mongoose and TypeScript. 

import type { AuthAdapter } from "kroxt/adapter";
import { User } from "./models.js";

export const mongoAdapter: AuthAdapter<any> = {
  createUser: async (data: any) => {
    const newUser = await User.create(data);
    return {
      id: newUser._id.toString(),
      ...newUser.toObject(),
    };
  },

  findUserByEmail: async (email: string) => {
    const user = await User.findOne({ email });
    if (!user) return null;
    return { id: user._id.toString(), ...user.toObject() };
  },

  findUserById: async (id: string) => {
    const user = await User.findById(id);
    if (!user) return null;
    return { id: user._id.toString(), ...user.toObject() };
  }
};
